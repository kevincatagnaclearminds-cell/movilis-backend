# Cómo descargar Alex Brush desde Google Fonts

## Opción 1: Descarga directa desde Google Fonts

1. Ve a: https://fonts.google.com/specimen/Alex+Brush
2. Haz clic en el botón **"Download family"** (botón con icono de descarga)
3. Se descargará un archivo ZIP
4. Extrae el ZIP y busca el archivo `AlexBrush-Regular.ttf`
5. Copia `AlexBrush-Regular.ttf` a esta carpeta: `src/public/fonts/`

## Opción 2: Usar Google Fonts Helper

1. Ve a: https://google-webfonts-helper.herokuapp.com/fonts/alex-brush
2. Selecciona "latin" (o el subset que necesites)
3. Descarga el archivo `.ttf`
4. Renómbralo a `AlexBrush-Regular.ttf` si es necesario
5. Colócalo en: `src/public/fonts/AlexBrush-Regular.ttf`

## Opción 3: Descarga directa (enlace)

Puedes descargar directamente desde:
- https://github.com/google/fonts/raw/main/apache/alexbrush/AlexBrush-Regular.ttf

## Verificar que funciona

Una vez que coloques el archivo, el código automáticamente:
- ✅ Detectará el archivo
- ✅ Lo cargará al generar PDFs
- ✅ Lo usará como fuente por defecto

**Nota:** El código ya está configurado para usar `AlexBrush-Regular.ttf` como fuente por defecto. Solo necesitas el archivo.
